📋 SETUP CHECKLIST (WHAT YOU NEED TO DO)
===============================================

STEP 1: Get Supabase Credentials
   □ Go to https://app.supabase.com
   □ Select your project
   □ Go to Settings → API
   □ Copy Project URL
   □ Copy anon (public) key
   See: GET_API_KEYS.md

STEP 2: Add to .env File
   □ Create/edit .env in project root
   □ Add VITE_SUPABASE_URL
   □ Add VITE_SUPABASE_ANON_KEY
   (See GET_API_KEYS.md for exact format)

STEP 3: Create Storage Bucket
   □ Go to Supabase Dashboard → Storage
   □ Create new bucket called: note-attachments
   □ Make it PUBLIC (for downloads)

STEP 4: Run App
   □ npm install (if not done)
   □ npm run dev
   □ App should start without errors

STEP 5: Test Authentication
   □ Navigate to landing page
   □ Click "Get Started"
   □ Sign up with test email
   □ Should redirect to dashboard
   □ Check Supabase dashboard → Users (should see your email)

STEP 6: Test Database
   □ On dashboard, create a note
   □ Go to Supabase → SQL Editor
   □ Run: SELECT * FROM notes;
   □ Should see your note

STEP 7: Test File Upload
   □ In Notes page, create note with file
   □ Go to Supabase → Storage → note-attachments
   □ File should appear (organized by user ID)

===============================================
📊 DATABASE STRUCTURE
===============================================

Core Tables:
  • profiles - User data (auto-created on signup)
  • notes - User workout notes
  • note_attachments - Files attached to notes
  • user_workout_logs - Completed workouts
  • progress_logs - Body measurements
  • streaks - Workout streak tracking
  • subscriptions - Premium tier

Content Tables:
  • workouts - Workout plans
  • exercises - Exercise library
  • nutrition_tips - Nutrition content
  • daily_challenges - Daily tasks
  • badges - Achievement definitions

Join Tables:
  • workout_exercises - Links workouts to exercises
  • user_badges - User achievement progress
  • saved_workouts - Bookmarked workouts
  • user_challenge_completions - Challenge tracking
